C_PICTURE_R.ERB ファイルは、もともと
C_PICTURE _R.ERB というファイル名のものです。
C_PICTURE と _R.ERB の間にスペースが入っていたので、
そのスペースを削りました。

同様に、
C_PICTURE_RD.ERB ファイルは、もともと
C_PICTURE _RD.ERB というファイル名のものです。

このスペースに何か意味があるのでしたら、
お手数ですが、作者さんの方で元に戻してください。

互換性を保つために、ダミーで
C_PICTURE _R.ERB
C_PICTURE _RD.ERB
という空のファイルを残してあります。
パッチではなく、フルでアップする際にダミーを削除すると良いかと思います。


同封した CAFE_STR.ERB は、ERB/都市関連/カフェ/ に置くべきかもしれません。

INFO_STR.ERB は、キャラの情報を表示するための、
