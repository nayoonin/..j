◇イベント発生方法◇
外界調査から【Lv0:都市境界線】の調査を行うと、確率で発生します。
他のレベルでは発生しないので注意。
発生条件にNTRモードつけるか迷いましたが、個人的にそういうのじゃないなと思うので無条件で。
選択肢で回避可能だしね…。

◆内容◆
都市境界線に居ついた野良犬の世話をしているはずがいつの間にか…。
聖女の子宮の有り無しで分岐します。有りで妊娠。
交配実験イベントの描写と矛盾してる気もしますが、まあこちらは自然な生物相手なのでということで…。
