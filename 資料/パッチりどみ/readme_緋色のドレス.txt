eraAS_0.58　追加遺物「緋色のドレス」
---------------------

・新規遺物の追加だよ、元ネタとか存在しない触手服だよ。
　　普通の触手服にしようとしたら何か物足りなくてやっちゃった感じだよ。
    サポーターがいない場合を考慮してない無能だよ、許して

・システム面で見ると経済的に優しくて、それなりの効率的だけど
　　실패율と데미지が大きいって感じにしたよ。こういうのはバランスが分からないね。
     肉体よりも精神に데미지がいくのは思考汚染してる結果的な設定も考えたけど意味ないね。

　純触手服は他の人に任せるよ、多少発想が被っても問題ないって信じてる。
